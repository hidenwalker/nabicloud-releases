나비구름 (NabiCloud)

libhangul 계열 Windows 한글 입력기(IME) — 세벌식·신세벌·두벌식 갈마들이·모아치기 등 다양한 자판을 지원합니다.

1. 다운로드

① 본체(먼저 설치) — 두벌식·세벌식 314·세벌식 최종 순아래(구름) 등 기본 자판 + 안마태· 신세벌 확장 세벌식 17종 + 한자 변환.

본체 설치 파일 받기NabiCloud-v*.exe · GitHub Releases

② 애드온(선택, 본체 설치 후) — 세벌식 3-18Na·명랑소녀 4z·세모이 (모아치기 e)·두겹이/두줄이(두벌식 모아치기).

애드온 설치 파일 받기NabiCloud-Addon-v*.exe · GitHub Releases

2. 설치 순서

  1. 본체 설치 파일을 실행합니다(관리자 권장). x64·x86 듀얼아키가 자동 설치되고 입력기 목록에 나비구름이 추가됩니다.
  2. 필요하면 애드온 설치 파일을 이어서 실행합니다(다중 선택 — 원하는 자판만 골라 설치).
  3. Win+Space(또는 언어 막대)에서 나비구름을 선택하면 바로 사용할 수 있습니다.

제거는 Windows 설정 → 앱 → 설치된 앱에서 "NabiCloud"(또는 애드온 이름) 제거.

3. 라이선스 오픈코어

  • 본체(TSF 셸·설정 UI·한자 변환·인스톨러) = 상용 proprietary — 소스 비공개.
  • 한글 조합 엔진 = 동적링크 libhangul.dll(LGPL-2.1) — 공개 소스: nabicloud-engine.
